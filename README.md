Инструкция по добавлению кейса. в АО. 
1. Заходим в нужный файл выше, каждый файл- список АО на такой то участок(Есть описания в названии)
2. Ctrl+F и в окне вставляем новое ключевое словосочетание. Если совпадений нет, это не дубликат
3. Изменяем файл, вставляя новое ключевое словосочетание
4. <img width="435" height="483" alt="image" src="https://github.com/user-attachments/assets/8e634bf4-189e-4bab-a5bf-810d42a89d99" />

Нажимая на commit changes видим следующее окно. В названии пишем тип изменения(добавление нового, удаление старого, предположим вы нашли дубль), список слов что добавили, и файл где произошли изменения.
В нижнем меню выбираем Создание НОВОЙ ВЕТКИ, дабы было удобнее смотреть/откатываться назад, и propose changes
<img width="846" height="629" alt="image" src="https://github.com/user-attachments/assets/c1024c79-45dc-4117-a684-1e9924925a90" />
Через стрелку выбираем Create pull request
 <img width="1293" height="875" alt="image" src="https://github.com/user-attachments/assets/abfe3efa-bb17-416f-8778-710410bca2dd" />
5.Жмём Merge pull request и дальше commit merge

6. Если вам нужно просто взглянуть на недавно добавленные АО, жмём Pull requests, и на плашку closed
